# Privacy Policy

## Data collection

This extension does **not** collect, store, or share personal data.

## Data transmission

This extension does **not** send data to external servers.

## Permissions usage

- `tabs`: used only to mute/unmute the current tab when ad indicators are detected.
- `storage`: used only to persist the user toggle state (enabled/disabled).
